🔥 Connect to Firebase

This one-time setup links your family app to a free Firebase database so everyone's data stays in sync across all devices.

  1. Go to console.firebase.google.com
  2. Click Add project → name it (e.g. "Our Family App") → Continue
  3. In your project, click Firestore Database → Create database → Start in test mode → Enable
  4. Click the ⚙️ gear → Project settings → scroll to "Your apps" → click </> (Web)
  5. Register app → copy the firebaseConfig object and paste it below
